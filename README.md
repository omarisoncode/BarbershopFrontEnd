# Frontend

React + Vite app for the public homepage, booking funnel, customer dashboard, and admin dashboard.

## Commands

- `npm run dev`
- `npm run build`
- `npm run lint`
- `npm run test`

## Environment

- `VITE_API_URL` points the frontend to the backend API
- REST requests and live booking updates both use the same `VITE_API_URL`
- Start from `frontend/.env.example`

See the root [README](../README.md) for the full project setup and shared scripts.
